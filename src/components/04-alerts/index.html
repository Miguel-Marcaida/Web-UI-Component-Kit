Módulo 04: Alertas y Mensajes

Componentes de alerta contextuales para comunicar mensajes importantes como éxito, error, o advertencia a los usuarios.

01. Alertas Base

Diseño fundamental de alertas con colores contextuales (éxito, info, warning, error).

02. Alertas con Cierre

Alerta base que incluye un botón para que el usuario pueda cerrarla o removerla de la vista.

03. Alertas con Encabezado

Variación de la alerta base que incluye un título destacado para jerarquizar el mensaje.

04. Alertas Solo Ícono

Variación minimalista que enfatiza el ícono y utiliza tipografía sutil para el mensaje.
