Módulo 21: Web Components

Patrones de encapsulación utilizando tecnologías nativas del navegador: Custom Elements y Shadow DOM. Se prioriza la **Autonomía Estricta** a nivel de arquitectura.

Subcomponentes del Módulo

01. Custom Element Base

Definición del Custom Element más simple con ciclo de vida y manejo de atributos.

02. Shadow DOM (Encapsulación CSS)

Componente utilizando Shadow DOM para el aislamiento total de estilos, ejemplificando la autonomía estricta.

03. Slots (Composición)

Uso de la etiqueta <slot> para permitir la inyección de contenido externo y composición del componente.
