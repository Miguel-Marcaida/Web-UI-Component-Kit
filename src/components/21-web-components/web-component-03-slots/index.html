21.03: Slots (Composición)

Demostración de la API de **Slots** para permitir la inyección de contenido externo (Light DOM) en puntos específicos dentro del Custom Element (Shadow DOM). Esto habilita la composición avanzada y la flexibilidad de diseño.

Este es el cuerpo principal del artículo. Utiliza el **slot por defecto** y permite al usuario definir la estructura de la sección central.

El Shadow DOM solo provee el marco (header/footer) y los estilos base.
