17.03: Estados y Propiedades ARIA

Este patrón ilustra el uso correcto de aria-expanded (para comunicar el estado de colapsado), aria-controls (para vincular el control con el contenido) y aria-disabled (para comunicar un estado inactivo).

Vista Previa: Control de Acordeón Accesible

Los estados ARIA comunican el estado actual de un elemento (ej: presionado, expandido, seleccionado) a las tecnologías de asistencia.

El botón superior no puede ser clickeado, y el lector de pantalla anuncia su estado.
