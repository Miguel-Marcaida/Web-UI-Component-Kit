17.02: Gestión de Foco (Focus Management)

Este patrón demuestra cómo gestionar el foco del teclado para crear una trampa de foco (Focus Trap) utilizando JavaScript y el uso correcto de tabindex="0" y tabindex="-1". Es esencial para modales y *sidebars*.

Vista Previa: Trampa de Foco en Panel

Active el panel y luego intente usar la tecla `Tab`. El foco debe quedar atrapado dentro del panel.
