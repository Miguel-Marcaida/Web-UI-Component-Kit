Módulo 17: Accesibilidad Avanzada (A11Y)

Patrones de marcado ARIA, gestión de foco y técnicas para lectores de pantalla, asegurando que todos los componentes sean utilizables por cualquier usuario.

01. Regiones Vivas (ARIA Live)

Patrón para actualizar contenido de forma dinámica (ej: notificaciones) usando aria-live sin interrumpir el flujo del lector de pantalla.

02. Gestión de Foco (Tabindex)

Establece reglas canónicas para la gestión del foco del teclado y la creación de tabindex para widgets complejos.

03. Estados y Propiedades ARIA

Guía de uso para propiedades como aria-expanded, aria-disabled y aria-controls en componentes interactivos.
