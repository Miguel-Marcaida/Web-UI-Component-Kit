Módulo 15: Layouts

Clases y patrones que controlan la disposición de los elementos en la página, utilizando Flexbox y Grid para crear estructuras robustas y responsivas.

01. Flexbox y Grid Helpers

Clases rápidas para inicializar contenedores Flex o Grid, controlando la alineación y el espaciado interno.

02. Stack y Box Models

Utilidades para apilar elementos vertical u horizontalmente (Stack) y para definir rápidamente dimensiones (ancho/alto) de contenedores.

03. Objeto Media (Media Object)

Patrón de disposición para una imagen fija (izquierda/derecha) junto a un bloque de contenido flexible. Ideal para comentarios o feeds.

04. Centrado (Center Utility)

Utilidad para centrar contenido horizontal y verticalmente dentro de cualquier contenedor, usando Flexbox.
