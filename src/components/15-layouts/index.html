Módulo 15: Layouts

Clases y patrones que controlan la disposición de los elementos en la página, utilizando Flexbox y Grid para crear estructuras robustas y responsivas.

01. Display y Utilidades (Flex/Grid)

Clases autónomas para inicializar contenedores Flex o Grid, controlando la dirección y la alineación.

02. Stack y Composiciones Verticales

Utilidades para apilar elementos verticalmente (Stack) y controlar el espaciado entre ellos de forma consistente.

03. Inline y Composiciones Horizontales

Clases para agrupar elementos en línea (`inline-group`) y controlar su espaciado horizontal.

04. Objeto Media y Centrado

Patrones para Media Object (imagen + contenido) y utilidades para centrado horizontal y vertical.
