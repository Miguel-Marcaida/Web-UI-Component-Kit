15.03: Objeto Media (Media Object)

Patrón de disposición para una imagen fija (izquierda/derecha) junto a un bloque de contenido flexible. Ideal para comentarios, avatares y listas de elementos.

1. Objeto Media Base (Alineación a la Izquierda)

Título del Contenido Flexible

Este es el cuerpo flexible. La imagen tiene un ancho fijo (ej. 64px), mientras que este bloque de texto ocupa el espacio restante de forma automática.

2. Objeto Media Inverso (Alineación a la Derecha)

Título del Contenido a la Derecha

Aplicando la clase .media--reverse, el elemento de imagen se moverá al final (derecha) del contenedor Flex.

CSS (Estilos de componente)


/* MEDIA OBJECT - CSS PRINCIPAL */
.media {
    display: flex; /* Contenedor Flex principal */
    align-items: flex-start; /* Alinea los ítems en la parte superior */
}
.media--reverse {
    flex-direction: row-reverse; /* Mueve la imagen al final */
}

.media-img {
    flex-shrink: 0; /* Asegura que la imagen no se encoja */
    width: 64px; /* Ancho fijo para la imagen/avatar */
    height: 64px;
    border-radius: 50%;
    object-fit: cover;
    background-color: var(--color-bg-light);
}

.media-body {
    flex-grow: 1; /* Permite que el cuerpo ocupe todo el espacio restante */
}
