15.02: Stack y Box Models

Clases auxiliares para crear listas de elementos apilados (Stack) con espaciado consistente y utilidades para controlar la altura y la anchura de los contenedores.

1. Utilidad Stack (Apilamiento)

Utiliza la clase .stack junto con una utilidad de espaciado vertical (m-y-3, por ejemplo) para añadir un espacio uniforme entre los elementos *internos*.
*Nota: Esta implementación requiere la utilidad `m-y-*` en los ítems internos.*

2. Dimensiones Fijas (Width)

Utiliza .w-* para establecer un porcentaje de ancho específico.

.w-25
.w-50
.w-75
.w-full (100%)
