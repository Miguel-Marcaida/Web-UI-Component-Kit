15.04: Centrado (Center Utility)

La utilidad .center-xy es un patrón **Flexbox** que permite centrar perfectamente un elemento horizontal y verticalmente dentro de su contenedor padre, sin importar las dimensiones del hijo.

1. Centrado Horizontal y Vertical (XY)

El contenedor tiene la clase .center-xy y una altura mínima para demostrar el centrado vertical.

¡Estoy Centrado!

2. Centrado Horizontal (Bloque)

Para centrar un elemento de bloque de ancho fijo, utiliza la utilidad de margen horizontal automático: .m-x-auto.

Centrado con m-x-auto

CSS (Estilos de componente)


/* CENTRADO FLEXBOX (Horizontal y Vertical) */
.center-xy {
    display: flex !important;
    justify-content: center !important; /* Centra horizontalmente */
    align-items: center !important; /* Centra verticalmente */
}

/* CENTRADO DE BLOQUE (Horizontal solamente) */
.m-x-auto {
    margin-left: auto !important;
    margin-right: auto !important;
}
