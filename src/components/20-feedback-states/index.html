Módulo 20: Feedback y Estados

Componentes visuales y patrones de interacción diseñados para ofrecer información clara al usuario sobre el **estado actual de la aplicación**, incluyendo carga, espera y ausencia de contenido.

Subcomponentes del Módulo

01. Loader Animado (Spinner)

Implementación de indicadores de carga accesibles (ARIA) y animados con CSS puro para una transición fluida.

02. Skeleton Screen

Patrón para simular la estructura de la interfaz durante la carga, mejorando la percepción de velocidad.

03. Empty State (Estado Vacío)

Diseño de interfaces para cuando no hay contenido que mostrar, incluyendo ilustraciones y llamados a la acción.
