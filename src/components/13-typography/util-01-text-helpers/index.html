Utilidades de Texto

Clases de ayuda de un solo propósito para modificar la alineación, el peso (bold/light), la transformación (mayúsculas/minúsculas) y la decoración del texto.

1. Alineación de Texto (`text-`)

Clases para la propiedad text-align.

Clase Propiedad CSS
text-start text-align: left
text-end text-align: right
text-center text-align: center

2. Transformación de Texto (`text-`)

Clases para la propiedad text-transform.

Este texto está en mayúsculas (text-uppercase).

ESTE TEXTO ESTÁ EN MINÚSCULAS (text-lowercase).

este texto tiene capitalización (text-capitalize).

3. Peso y Estilo de Fuente (`fw-`, `fst-`)

Clases para el peso (font-weight) y el estilo (font-style) de la fuente.

Texto en negrita (fw-bold).

Texto con peso ligero (fw-light).

Texto en cursiva (fst-italic).

4. Código de Utilidades CSS (Para /CSS/main.css)

Estas clases son de ámbito global y deben ser añadidas al archivo **/CSS/main.css**.
