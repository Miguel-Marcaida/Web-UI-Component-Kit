Tipografía 01: Jerarquía Base

Definición de estilos base para todos los títulos (h1 a h6) y párrafos.

H1: El título principal más grande (2.5rem)

Usado para el título de página o sección principal.

H2: Título de sección primaria (2rem)

Define la estructura principal del contenido.

H3: Subsección importante (1.75rem)

Usado para listar elementos dentro de una sección H2.

H4: Encabezado de componente o lista (1.5rem)

A menudo usado en títulos de tarjetas o formularios.

H5: Título pequeño de bloque (1.25rem)

Útil para detalles o información secundaria.

H6: El título más pequeño (1rem)

Usado para etiquetas o metadatos.
