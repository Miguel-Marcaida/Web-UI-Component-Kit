16.01: Toggle y Eventos Básicos

La función central para manejar la interactividad es toggleState(), la cual simplemente alterna una clase (ej. .is-active) en un elemento objetivo cada vez que se hace clic en un disparador.

1. Toggle de Clase Simple

El botón (disparador) alterna la clase .is-active en el panel (objetivo).

Este panel cambia su apariencia al recibir la clase .is-active.

/* ----------------------------------------------------- */
/* UTILITY: TOGGLE STATE (Implementación Base en main.js) */
/* ----------------------------------------------------- */

function toggleState(event) {
    // 1. Evita la acción por defecto (si es un enlace, etc.)
    event.preventDefault();

    // 2. Obtiene el disparador (el elemento en el que se hizo clic)
    const trigger = event.currentTarget;

    // 3. Obtiene el ID del objetivo a partir del atributo data
    const targetId = trigger.dataset.toggleTarget;
    if (!targetId) {
        console.error("Falta data-toggle-target en el disparador.");
        return;
    }
    
    // 4. Busca el elemento objetivo en el DOM
    const targetElement = document.getElementById(targetId);
    if (!targetElement) {
        console.error(\`Elemento objetivo con ID \${targetId} no encontrado.\`);
        return;
    }

    // 5. Alterna la clase .is-active en el elemento objetivo
    targetElement.classList.toggle('is-active');

    // 6. Opcional: Alterna la clase en el propio disparador para retroalimentación visual
    trigger.classList.toggle('is-active');
}

// -----------------------------------------------------
// INICIALIZACIÓN (Dentro de una función DOMContentLoaded)
// -----------------------------------------------------

document.addEventListener('DOMContentLoaded', () => {
    // 1. Selecciona todos los elementos que tienen el atributo data-toggle-target
    const toggles = document.querySelectorAll('[data-toggle-target]');

    // 2. Asigna el event listener a cada uno
    toggles.forEach(toggle => {
        toggle.addEventListener('click', toggleState);
    });
});
