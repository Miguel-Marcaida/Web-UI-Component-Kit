16.02: Debounce y Throttle

Estas utilidades controlan la frecuencia de ejecución de funciones ligadas a eventos rápidos (como escribir, mover el mouse, o redimensionar), mejorando significativamente el rendimiento.

1. Debounce (Para búsqueda)

La función se ejecuta **solo después** de que el usuario deja de escribir por 300ms.
