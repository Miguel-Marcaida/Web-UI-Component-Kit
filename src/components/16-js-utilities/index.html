Módulo 16: JavaScript Utilities

Funciones y lógica de JavaScript reutilizable para manejar la interactividad, el estado (toggle) y la manipulación del DOM, dando vida a los componentes del kit.

01. Toggle y Eventos Básicos

Lógica fundamental para alternar clases (ej. .is-active) y controlar la visibilidad de elementos con un solo clic.

02. Debounce y Throttle

Funciones de alto rendimiento para limitar la frecuencia de ejecución de eventos (resize, scroll, keyup, etc.).

03. Eventos Personalizados (Custom Events)

Mecanismo para que los módulos se comuniquen de forma desacoplada (ej. notificar que un Modal se ha cerrado).
