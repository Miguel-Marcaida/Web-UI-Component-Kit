16.01: Transición Canónica (Base)

Este patrón establece las variables CSS canónicas para la duración (`--transition-duration-md`) y la curva de aceleración (`--transition-timing-ease`), garantizando la coherencia de movimiento en todos los componentes autónomos del Kit.
