Módulo 16: Animaciones y Transiciones

Patrones canónicos y utilidades CSS para gestionar el movimiento, la suavidad de las interacciones y la accesibilidad de las animaciones (`prefers-reduced-motion`).

01. Transición Canónica (Base)

Define variables CSS para duraciones y curvas de movimiento, y demuestra la aplicación de transiciones de bajo costo.

02. Keyframes y Clases

Catálogo de animaciones CSS (`@keyframes`) reutilizables (fade-in, spin, etc.) para copiar en componentes autónomos.

03. Preferencia de Movimiento

Implementación de `prefers-reduced-motion` para garantizar la accesibilidad y reducir el movimiento excesivo.
