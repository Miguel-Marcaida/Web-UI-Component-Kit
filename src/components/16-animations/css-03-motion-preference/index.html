16.03: Preferencia de Movimiento (`prefers-reduced-motion`)

Este patrón utiliza el *media query* @media (prefers-reduced-motion: reduce) para deshabilitar o simplificar animaciones complejas, garantizando la accesibilidad (A11y) y previniendo el mareo o la fatiga visual.

Vista Previa: Animación Condicional

La caja se mueve lateralmente de forma continua. Esta animación debe detenerse si la preferencia de "Reducir movimiento" está activa en su sistema operativo/navegador.
