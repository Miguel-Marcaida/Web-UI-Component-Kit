Módulo 10: Componentes de Navegación de Página

Elementos esenciales para ayudar al usuario a navegar grandes volúmenes de contenido (Paginación) y a entender su ubicación dentro de la estructura del sitio (Breadcrumbs).

01. Paginación Base

Controles estándar para navegación de grandes listas de resultados.

02. Paginación Compacta

Diseño minimalista o basado en puntos, útil para carruseles y navegaciones pequeñas.

03. Migas de Pan (Breadcrumb) Base

Rastreador de la ubicación actual del usuario dentro de una jerarquía de navegación.

04. Migas de Pan con Iconos

Mejora la legibilidad de la jerarquía añadiendo iconos descriptivos.
