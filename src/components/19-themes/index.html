Módulo 19: Temas y Esquemas de Color

Este módulo aborda la gestión de esquemas de color, la implementación robusta del Modo Oscuro, y la personalización de marca utilizando Variables CSS y las consultas de medios (`@media`) nativas del navegador.

Subcomponentes del Módulo

01. Modo Oscuro (Persistencia)

Implementación del tema oscuro basado en la preferencia del usuario y con almacenamiento local (localStorage).

02. Personalización (Color de Marca)

Uso de variables CSS para permitir a los usuarios definir un color primario de acento y marca.
