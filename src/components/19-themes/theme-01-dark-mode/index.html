19.01: Modo Oscuro (Dark Mode)

Demuestra el sistema canónico para implementar el tema oscuro, respetando la preferencia del sistema operativo (`prefers-color-scheme`) y permitiendo al usuario sobrescribir y **guardar su elección** mediante un *toggle* (persistente vía localStorage).
