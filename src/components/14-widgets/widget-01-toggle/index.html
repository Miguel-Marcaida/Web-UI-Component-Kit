14.01: Toggle de Contenido (Colapsable y Acordeón)

Este componente maneja la funcionalidad de Colapsable (múltiples paneles abiertos) y Acordeón (solo un panel abierto) mediante un único mecanismo con JavaScript autónomo.

Vista Previa: Modo Acordeón (Solo uno abierto a la vez)

Nuestra política de privacidad garantiza que sus datos personales nunca serán compartidos con terceros sin su consentimiento explícito. Usamos cifrado de extremo a extremo para toda la información sensible.

Al utilizar este servicio, usted acepta que es responsable de mantener la confidencialidad de su cuenta y contraseña y de restringir el acceso a su dispositivo.

Este widget es ideal para preguntas frecuentes. Si necesita el modo "Colapsable" (abrir varios a la vez), simplemente elimine el atributo data-mode="accordion" del contenedor superior.

Referencia de Nomenclatura y Código Fuente

El código fuente es totalmente autónomo e incluye su propio script para la funcionalidad. El CSS y el JavaScript deben copiarse junto con el HTML.
