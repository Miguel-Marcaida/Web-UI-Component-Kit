Módulo 14: Widgets Avanzados

Componentes interactivos complejos que han sido **re-arquitecturados** para garantizar la **Autonomía Estricta**. Cada widget incluye su propio CSS y JavaScript local, sin depender de archivos centrales del Kit.

01. Toggle (Colapsable/Acordeón)

Mecanismo base para mostrar/ocultar contenido, con modo configurable de colapsable simple o acordeón.

02. Dropdown / Popover

Muestra contenido flotante que se cierra al hacer clic fuera (manejo de estado global con JS autónomo).

03. Tabulador (Tabs)

Navegación por paneles de contenido con lógica de grupo gestionada por su propio script.
