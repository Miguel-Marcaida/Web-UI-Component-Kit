Módulo 18: Patrones de Interacción (JS)

Funciones JavaScript puras y reutilizables para optimizar el rendimiento y la experiencia de interacción, como la gestión del flujo de eventos (Debounce, Throttle) y utilidades de DOM.

01. Debounce (Retraso de Llamada)

Patrón para optimizar la ejecución de eventos repetitivos (ej: búsqueda en input) asegurando que la función se ejecute solo una vez tras un periodo de inactividad.

02. Throttle (Limitador de Tasa)

Patrón que limita la ejecución de una función a una tasa máxima fija (ej: una vez cada 200ms), crucial para eventos de scroll o resize.

03. Observadores (Intersection/Mutation)

Uso eficiente de las API de Observadores nativas para la carga diferida (Lazy Loading) y la detección de cambios en el DOM sin impacto en el rendimiento.
