Nav 02: Menú Hamburguesa Interactivo (Mobile First)

Un componente de navegación esencial con un diseño moderno "Mobile First". Implementa un mecanismo de "menú hamburguesa" interactivo para pantallas pequeñas, convirtiéndose en una barra de navegación horizontal completa en escritorio. Ideal para aplicaciones web, asegurando la accesibilidad y la fluidez del usuario.
