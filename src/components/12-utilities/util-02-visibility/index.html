Utilidades de Visualización y Flujo

Clases de ayuda para modificar rápidamente la propiedad display, el flujo de elementos (`float`) y la visibilidad para lectores de pantalla.

1. Utilidades de Display (`d-`)

Clases para cambiar el valor de la propiedad display. Útil para modificar rápidamente cómo se comporta un elemento sin editar el CSS del componente.

Clase Propiedad CSS
d-none display: none
d-inline display: inline
d-block display: block
d-inline-block display: inline-block
d-flex display: flex
d-inline-flex display: inline-flex

2. Utilidades de Flotación (`float-`)

Clases para flotar elementos a la izquierda o derecha, o para eliminar la flotación.

Clase Propiedad CSS
float-start float: left
float-end float: right
float-none float: none
clearfix Soluciona el problema de elementos flotantes.

3. Utilidades de Lectores de Pantalla

Clases para ocultar visualmente un elemento **mientras se mantiene disponible para tecnologías de asistencia** (ej. lectores de pantalla).
