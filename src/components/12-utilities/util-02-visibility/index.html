Utilidad 02: Visibilidad

Clases de utilidad para modificar la propiedad display o asegurar que el contenido sea accesible, incluso si está oculto visualmente.

Clase: .d-none

Oculta visualmente y estructuralmente el elemento (display: none !important;). No afecta el layout.
