Utilidades de Espaciado (Margin y Padding)

Clases de utilidad para modificar rápidamente los márgenes (`margin`) y los paddings (`padding`) de los elementos. El sistema se basa en una escala predefinida y es **responsivo**.

1. Nomenclatura del Sistema

Las clases siguen el formato propiedad-lado-tamaño, donde:

  • **Propiedad:** m para margin y p para padding.
  • **Lado:**
    • t: top (arriba)
    • b: bottom (abajo)
    • s: start (izquierda - para LTR)
    • e: end (derecha - para LTR)
    • x: left y right (eje horizontal)
    • y: top y bottom (eje vertical)
    • (vacío): todos los lados
  • **Tamaño:** Un número del 0 al 5 basado en el sistema de escala.

2. Escala de Espaciado

La escala de tamaño utiliza variables CSS, asumiendo un tamaño de espaciado base de **1rem** (generalmente 16px).

Clase de Tamaño Valor CSS (Ejemplo)
0 0
1 0.25rem
2 0.5rem
3 1rem
4 1.5rem
5 3rem
auto auto (Solo para márgenes)
