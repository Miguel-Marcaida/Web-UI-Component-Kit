Multimedia 02: Objeto Multimedia (Media Object)

Un patrón de diseño modular y flexible, ideal para listar elementos que combinan una imagen o avatar (izquierda) con contenido textual asociado (derecha). Utiliza Flexbox para el alineamiento.

Vista Previa

Título del Objeto Multimedia (Por defecto)

Este es el cuerpo principal del contenido. El componente se alinea perfectamente usando Flexbox, sin necesidad de flotantes. El cuerpo toma el espacio restante.

Publicado hace 3 horas.

Objeto Multimedia (Invertido)

Esta variación muestra la imagen o figura alineada a la derecha, invirtiendo el orden de los elementos mediante la clase .is-reversed de Flexbox.

Comentario reciente.

Objeto Multimedia (Centrado)

La figura se alinea al centro verticalmente, gracias a la propiedad align-items: center; de Flexbox, útil para contenido de una sola línea.
